# group delete
Deletes a resource group.

## Options
* `-s, --subscription-id` - (Required) (Required) Subscription ID.
* `-n, --name` - (Required) (Required) Resource group name.

## Examples

### Delete a resource group
```bash
$ topaz group delete \
    --name "my-rg" \
    --subscription-id "6B1F305F-7C41-4E5C-AA94-AB937F2F530A"
```
