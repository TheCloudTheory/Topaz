# subscription enable
Enables a subscription, setting its state to Enabled.

## Options
* `-i, --id` - (Required) (Required) Subscription ID.

## Examples

### Enable a subscription
```bash
$ topaz subscription enable \
    --id "6B1F305F-7C41-4E5C-AA94-AB937F2F530A"
```
