# cosmosdb account list-by-subscription
Lists all Azure Cosmos DB accounts in a subscription.

## Options
* `-s, --subscription-id` - (Required) (Required) Subscription ID.

## Examples

### List Cosmos DB accounts in a subscription
```bash
$ topaz cosmosdb account list-by-subscription \
    --subscription-id "00000000-0000-0000-0000-000000000000"
```
