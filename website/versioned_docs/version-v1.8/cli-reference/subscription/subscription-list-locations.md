# subscription list-locations
Lists all available Azure locations for a subscription.

## Options
* `-i, --id` - (Required) (Required) Subscription ID.

## Examples

### List subscription locations
```bash
$ topaz subscription list-locations \
    --id "6B1F305F-7C41-4E5C-AA94-AB937F2F530A"
```
