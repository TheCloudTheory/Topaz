# subscription cancel
Cancels a subscription, setting its state to Disabled.

## Options
* `-i, --id` - (Required) (Required) Subscription ID.

## Examples

### Cancel a subscription
```bash
$ topaz subscription cancel \
    --id "6B1F305F-7C41-4E5C-AA94-AB937F2F530A"
```
