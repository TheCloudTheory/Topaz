# subscription create

Creates a new subscription.

## Options
* `-i|--id` - (Required) subscription ID
* `-n|--name` - (Required) subscription name

## Examples

### Create a subscription
```bash
$ topaz subscription create --id 36a28ebb-9370-46d8-981c-84efe02048ae --name "sub-local"
```
