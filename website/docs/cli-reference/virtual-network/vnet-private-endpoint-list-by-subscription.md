# vnet private-endpoint list-by-subscription
Lists private endpoints in a subscription.

## Options
* `-s, --subscription-id` - (Required) subscription ID

## Examples

### Lists all private endpoints in a subscription
```bash
$ topaz vnet private-endpoint list-by-subscription --subscription-id 36a28ebb-9370-46d8-981c-84efe02048ae
```
